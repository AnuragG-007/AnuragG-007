### üéÆ Steam GameFinder (LLM, Vector Search & Backend)
>*An AI-powered semantic game discovery and recommendation system for Steam.*

This project demonstrates my ability to design production-style AI systems, combining LLM reasoning, vector search, and hybrid ranking to solve real-world discovery problems.

üß† Intent Understanding: LLaMA 3.1‚Äìbased semantic parsing for mood, genre, and gameplay mechanics.

üîé Retrieval: Dense semantic embeddings (multilingual-e5-large) with Pinecone vector search.

‚ö° Ranking Engine: Hybrid re-ranking using semantic similarity, popularity heuristics, quality metrics, and penalties.

ü§ñ Reasoning Layer: Backend-level explainability ensuring logically consistent recommendations.

‚öôÔ∏è Backend: Scalable FastAPI service with clean, frontend-agnostic JSON APIs.

üíª Frontend: Interactive UI integrated with backend APIs (Next.js-based workflow).
